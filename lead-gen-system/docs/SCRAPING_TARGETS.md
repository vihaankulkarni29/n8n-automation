## üîß HOW TO SCRAPE ANY CATEGORY

### Step 1: Find the NCT Code
1. Go to JustDial.com
2. Search for your category (e.g., "Bakeries in Mumbai")
3. Copy the URL - it will have format: `/Mumbai/Category-Name/nct-XXXXXXXX`
4. The `nct-XXXXXXXX` is your category code

### Step 2: Update Scraper
Edit `scrapers/detailed_justdial_scraper.py` line 532:
```python
url = "https://www.justdial.com/[CITY]/[CATEGORY]/nct-[CODE]"
```

### Step 3: Run Scraper
```bash
python scrapers/detailed_justdial_scraper.py
```

### Step 4: Analyze Results
```bash
python quick_data_helper.py analyze data/detailed_[category].csv
```

## üí° BEST PRACTICES

### Target Selection Criteria:
1. **High Volume**: Categories with 1,000+ businesses
2. **Low Digital Presence**: 50%+ have no website/social media
3. **High Value**: Businesses that can afford marketing services
4. **Urgent Need**: Industries being forced online (COVID effect)

### Geographic Strategy:
- **Tier 1 Cities** (Mumbai, Delhi, Bangalore): More businesses, higher budgets
- **Tier 2 Cities** (Pune, Ahmedabad, Hyderabad): Less competition, eager for digital
- **Mix Both**: 70% Tier 1, 30% Tier 2 for best conversion

### Scraping Efficiency:
- Each scrape: ~10-15 businesses per page
- Speed: ~2-3 minutes per page
- **Target**: 100 businesses in 20-30 minutes
- **Daily Goal**: 150 businesses = 45-60 minutes of scraping
